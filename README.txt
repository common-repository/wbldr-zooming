Plugin pozwala na wyświetlanie zdjęć w formie modala.

== Description ==

Wszystkie zdjęcia będące linkiem zamiast otwierać się w nowej stronie stają się modalem.

== Installation ==

1. Wrzuć `wbldr-zooming.php` do `/wp-content/plugins/` 
2. Aktywuj plugin w zakładce 'Pluginy' w WordPress
3. Wszystkie ustawienia znajdują się w 'Ustawienia/wbldr Zooming'
